# PLANILHAS (mínimo para Render)

## Como publicar
1) Suba estes arquivos no **GitHub** (um único repositório).
2) No **Render**: New → Web Service → conecte o repositório.
3) Plano **Free**. O Render lê o `Procfile` e inicia.
4) Abra o link do app.

Arquivos:
- `app.py`  (seu painel)
- `requirements.txt`
- `Procfile`
- `render.yaml`
- `README.md`
